Justice.js
==================================================
The goal of this project it to provide insight into the performance of a webpage.


#### Core Values:
  - easily embeddable
  - No dependencies
  - As small as possible
  - Render itself at 60fps or greater


#### Core Functionality:
  - Load times
  - FPS
  - works in modern browsers + graceful failure


#### Roadmap
  - console.log'ish proxy
  - window on error log
  - bookmarklet
  - cdn.js release
  - npm release
  - chrome plugin

#### Release versioning
  - `<major>.<minor>.<patch>`, ex: 0.1.8
